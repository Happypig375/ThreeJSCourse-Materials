1. Button, onclick and innerHTML

Add the following button in the first div:

<button type="button" onclick="btn1Clicked(this)">I'm Button 1, Click me!</button>

When this button is clicked, the btn1Clicked() custom function will be called, and this button is passed as the argument of this function call.

At the bottom of your code, right above </body>, add the following code segment:

<script>
    function btn1Clicked(elem) {
        console.log("console output");
        elem.innerHTML = "Don't click me!";
    };
</script>

Here, elem refers to the button that has been clicked. By changing its innerHTML, we replace I'm Button 1, Click me! with Don't click me!.

Very often, we'd like to print some debug messages to the console, this could be done by calling console.log(). In Chrome, we have to bring up the Developer tools in order to see the console output. Hit the button next to the search bar, choose More tools > Developer tools. Also, you can use shortcut ⌘ + Shift + I on macOS, or Ctrl + Shift + I on Windows to bring up the console.

2. Modifying Style

For the second div element, please add another button there:

<button type="button" onclick="btn2Clicked(this)">I'm Button 2, Click me!</button>

Nothing new, and here's the corresponding function, append it to the <script> area:

function btn2Clicked(elem) {
    elem.style = "display:none";
    elem.parentNode.innerHTML = "It's gone";
};

Line 2 changes the CSS style of the active element (i.e. the button). Setting display:none will hide the element. Line 3 basically has similar effect, as it replaces the button with a text string. Please note that the parentNode refers to outer div.

3. Modifying the Class

Another button for div03:

<button type="button" onclick="btn3Clicked(this)">I'm Button 3, Click me!</button>

This button will change the class for div03. We first have to set up a CSS rule in the <head> section:

<style>
    .class3 {
        background-color: lightsalmon
    }
</style>

In the <script> section, append the following function there:

function btn3Clicked(elem) {
    elem.parentNode.className += "class3";
};

Now, hitting the third button will give div03 a color background.

5. Basic JavaScript Types and typeof

JavaScript doesn't rely on buttons. Let's say if we want to execute some JavaScript functions when all the page contents (including images, script files, CSS files, etc.) have been loaded, this could be done by adding an onload attribute to the <body> tag:

<body onload="pageLoaded()">

Next, develop this function in the <script> area:

function pageLoaded() {
    runDiv5();
}

runDiv5() is our new custom function, as defined below:

function runDiv5() {

    var booleanVariable = true;
    var numericVariable = 12;
    var stringVariable = "Twelve";

    document.getElementById("div05").innerHTML = booleanVariable + " " + numericVariable + " " + stringVariable + "<br>";
    document.getElementById("div05").innerHTML
        += typeof booleanVariable + " " + typeof numericVariable + " " + typeof stringVariable;
}

Here, we define three variables with types: boolean, number and string. Won't go too deep here, as i believe it's been covered last semester. Just want to point out the typeof operator could be very useful.

6. JavaScript Object

JavaScript objects are simply containers for named values.

Let's define a JavaScript object and develop a loop to print out all the properties.

function runDiv6() {

    var sukiyaki = { beef: "2 lb", eggs: 4, crucial: "pot" };

    var string = "";

    for (item in sukiyaki) {
        string += item + ": " + sukiyaki[item] + "<br>";
    }
    
    // items are the keys (or names, indices), while sukiyaki[item] are the values.

    document.getElementById("div06").innerHTML = string;

}

Notice the use of for..in in line 7. Some may prefer a tabular layout, perform the required changes as follows:

var string = "<table>";

for (item in sukiyaki) {
    string += "<tr><td>" + item + "</td><td>" + sukiyaki[item] + "</td></tr>";
}

string += "</table>"

7. JavaScript Array

JavaScript array also belongs to JavaScript object. Let's develop an array and use another loop to print out its contents:

function runDiv7() {

    var sukiyakiParticipants = ["Martin", "Kenny", "Jarvis"];

    var string = typeof (sukiyakiParticipants) + "<br>"; // Object

    for (person of sukiyakiParticipants) {
        string += person + "<br>";
    }

    document.getElementById("div07").innerHTML = string;

}

Line 5 prints the type of this array and the result is object as it's also a container of indexed values. For array, it is easier to use for..of to output its elements.

An improved version is to use the unordered list format, as shown below:

var string = typeof (sukiyakiParticipants) + "<br><ul>";

for (person of sukiyakiParticipants) {
    string += "<li>" + person + "</li>";
}

string += "</ul>"

9. Conditional Statements

Very often when we write code, we want to perform different actions for different decisions. This could be implemented with the If..then..else structure:

function runDiv9() {

    if (Math.random() > 0.5) {
        document.getElementById("div09").innerHTML = "Upper";
    } else {
        document.getElementById("div09").innerHTML = "Lower";
    }
    
}

Here, Math.random() will give us a random number within 0 and 1. Line 3 perform a condition testing. If the result is true, line 4 will be executed; otherwise we will run line 6 instead.

10. For Loop

For loop may be quite complicated at first sight, but it's pretty handy to perform similar task repeatedly.

function runDiv10() {

    for (i = 0; i < 10; i = i + 1) {

        document.getElementById("div10").innerHTML += i + " ";

    }

}

i is a counter variable, initially set as 0. For each round, we check whether the condition could still be met (i < 10). So it could be met, we will execute the code block and update the counter (i = i + 1). Whole thing will repeat until the condition can no longer be met.

11. Date

12. SubString and Split

We may want to obtain a certain substring in a string. It could be easily done if the substrings are separated by some delimiters. If so, we can use the split() JavaScript function.

function runDiv12() {

    var str = "How are you doing today?";
    var res = str.split(" ");
    document.getElementById("div12").innerHTML = res[3]; // doing

}

The split() function expects the delimiter as its input argument. The result of this function is an array of substrings.
